# cpp-project-template

A simple C++ project template with [Abseil](https://abseil.io/) and
[GoogleTest](https://github.com/google/googletest).

## How to Build

**CMake**
```
./cmake_build.sh
```

**Bazel**
```
./bazel_build.sh
```

## How to Run

**CMake**
```
./cmake_run.sh
```

**Bazel**
```
./bazel_run.sh
```

## How to Test

**CMake**
```
./cmake_test.sh
```

**Bazel**
```
./bazel_test.sh
```
